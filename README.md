# 🎨 JS Simple Theme Changer

A lightweight and beginner-friendly JavaScript project that allows users to switch between light and dark themes dynamically.

---

## 🚀 Features

* 🌗 Toggle between Light and Dark mode
* 💾 Saves user preference using localStorage
* ⚡ Fast and simple implementation
* 🧩 Easy to integrate into any project
* 🎯 Beginner-friendly code structure

---

## 📂 Project Structure

```
JS-simple-ThemeChanger/
│── index.html
│── style.css
│── script.js
```

---

## 🛠️ How It Works

This project uses JavaScript to:

* Detect user interaction (button click)
* Toggle CSS classes for theme switching
* Store selected theme in `localStorage`

This ensures that the theme preference remains even after refreshing the page.
